
## Параметри
Відкрийте <i>File / Preferences / Settings / Extensions / B1 Extension</i>.
Встановіть такі параметри username і password, як при тестуванні.

## Вхід
- Переконайтеся, що відкрита та папка, в якій ви хочете зберігати свої розв'язки задач.  
- Виконайте команду LOGIN (з меню або зі списку команд, якщо меню не видно)
- Введіть повну назву задачі в форматі <i>задачник.задача</i> 

## Вирішення
У вікні, що відкриється, пишіть код и відправляйте на перевірку командою CHECK.

Перевірятися буде лише те, що знаходиться між коментарями BEGIN і END.<br>
Якщо таких коментарів немає, перевірятися буде увесь зміст вікна редактора.

Результат перевірки очікуйте у повідомленні (знизу справа).<br>
Разом з результатом перевірки прийде повідомлення про залишок часу на вирішення.
