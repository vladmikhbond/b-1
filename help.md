# Довідка

## Вхід
- Переконайтесь, що у VS Code відкрита та папка, в якій ви хочете зберігати свої вирішення задач.  
- Ctrl-Shift-P - відкрити список команд
- Виконати команду b1:Login 
- Ввести повну назву задачі в форматі <i>задачник.задача</i>

## Вирішення
У вікні, що відкриється, пишіть код и відправляйте на перевірку командою CHECK.

Перевірятися буде лише те, що знаходиться між коментарями BEGIN і END.
Якщо таких коментарів немає, перевірятися буде увесь зміст вікна редактора.

Результат перевірки очікуйте у повідомленні (знизу справа).

Разом з результатом перевірки прийде повідомлення про решту часу.
